  • id는 유일하게 사용해야하고, 상속해서 써야한다면 class로 정의하자.

    • Size Defintion


    • px: 고정값으로 표현
    • em: 부모의 배수로 표현
    • rem(root em): body(root)의 배수로 표현
    • %: 부모의 크기에 대한 비율
    • vw(viewport width): body(root) width에 대한 비율
    • vh(viewport height): body(root) height에 대한 비율

    Space


    margin: 50px, padding: 20px and box-sizing: border-box

    margin: 50px, padding: 20px and box-sizing: border-box

    • padding: 개체의 안쪽 여백을 다룸.
    • margin: 개체의 바깥쪽 여백을 다룸. (특히 개체와 개체 사이의 간격)
    • border-box: 해당 속성을 사용하는 경우 border를 포함한 size를 기준으로 width와 height을 결정한다.
