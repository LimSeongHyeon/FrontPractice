Simple Text

웹 개발은 디지털 세상을 만드는 열쇠입니다.
HTML로 웹 페이지 구조를 정의하고, CSS로 스타일을 입히며, JavaScript로 동적 기능을 추가합니다.
웹 개발자가 되기 위해선 끊임없이 학습하고, 프로젝트를 진행하며 경험을 쌓아야 합니다.
프론트엔드와 백엔드 개발, 데이터베이스, 보안 등 다양한 영역을 다루며, 사용자 경험을 개선하는 것이 주요 목표입니다.
웹 개발은 끊임없이 변화하며 창의적인 문제 해결 능력을 요구합니다. 하지만 그만큼 보상도 크며, 성장과 개발 가능성은 무한합니다.

Media(Image, Video)

alt = alternative text

  • 이미지 대체 텍스트: alt 속성은 웹 페이지에 이미지를 표시하지 못하는 경우(이미지 로드 실패, 이미지를 지원하지 않는 텍스트 브라우저, 스크린 리더 등)에 이미지의 대체 텍스트로 사용됩니다. 대체 텍스트는 시각적으로 이미지를 확인할 수 없는 사용자에게 이미지의 내용이나 의미를 설명해줍니다.

  • 웹 접근성: alt 속성은 웹 접근성을 향상시키는 데 중요한 역할을 합니다. 시각 장애인이나 저시력자, 텍스트 브라우저 사용자 등에게 웹 콘텐츠를 더욱 접근 가능하게 만들어줍니다.

  • 검색 엔진 최적화(SEO): 검색 엔진은 웹 페이지의 이미지를 이해하지 못합니다. alt 속성을 통해 이미지에 관한 설명을 제공하면 검색 엔진은 이미지를 더 잘 이해하고 콘텐츠를 인덱싱할 때 고려합니다. 이로써 이미지 검색 및 웹 페이지의 SEO에 도움이 됩니다.


srcset 속성을 이용하여 사진의 크기를 조정할 수 있다.



iframe을 이용하여 동영상도 보여줄 수 있음. height이 없으면 자동으로 계산함. (width 증감소율에 따라 조정.) audio나 video Tag를 활용하여 사용할 수 있음.
